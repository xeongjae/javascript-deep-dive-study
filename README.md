## 📚 스터디 진행방식
- 기본적으로 책 1장에 1주씩 진행.
- 미팅 전까지 이해한대로 요약한 내용, 미팅에서 나눌 이야기 정리해서 PR 올리기.
- 수요일 오후 9시 미팅.
- 여러 경험과 이야기를 나누는 방식을 지향.
- 매일 랜덤으로 뽑은 진행자가 진행.
